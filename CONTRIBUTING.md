# Contributing to JEP-Agent SDK

## Setup

```bash
git clone https://github.com/hjs-spec/jep-agent-sdk.git
cd jep-agent-sdk
make install
```

## Testing

```bash
make test
```

## Code Style

```bash
make format
make lint
```

## Pull Request Process

1. Ensure tests pass (`make test`)
2. Update examples if API changes
3. Update `CHANGELOG.md`
```
